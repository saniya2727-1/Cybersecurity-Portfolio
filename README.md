## 📎 Accessing Portfolio Documents

Most documents in this portfolio are stored as `.docx` files using [Git Large File Storage (Git LFS)](https://git-lfs.com/). GitHub does not support previewing `.docx` files directly in the browser.

To view the content:

- Click the **"View raw"** link under the file name
- The file will automatically download to your device
- Open with any Word processor (e.g., Microsoft Word or Google Docs)

> ⚠️ If cloning this repository and want to access all documents locally, make sure Git LFS is installed:  
> `git lfs install`
